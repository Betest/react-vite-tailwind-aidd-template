# Frontend Agent Playbook

## Component workflow
1. Identify responsibility
2. Define props + types
3. Implement minimal JSX
4. Apply Tailwind styles
5. Add icons (lucide-react)
6. Verify build/UI

## Refactors
- One component at a time
- Preserve public props
- No behavior change unless requested

## When stuck
- Reduce scope
- Request exact error output
- Stop after 3 failures and propose options
