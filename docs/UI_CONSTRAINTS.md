# UI Constraints

## Allowed
- React 19 + hooks + functional components
- TypeScript
- Vite
- TailwindCSS
- lucide-react (ALL icons & logos)

## Forbidden unless explicitly requested
- UI kits/design systems (MUI/Ant/Chakra/etc.)
- Other icon libraries or copied external SVGs
- CSS-in-JS libraries, theming libs
- New state/query libs

## Icons
- Always import from lucide-react
- Prefer semantic icons for actions
- No SVGs copied from external sources
- Use icons consistently for actions/branding
